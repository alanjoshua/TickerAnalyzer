A simple website written in Go that performs risk analysis and DCF for a given Ticker. 
This is mainly a side project for me to learn the Go programming language, and also experiment with some of the stuff that I have been learning as part of my CFA prep. 
I also heavily relied of Gemini for the frontend HTML and CSS since I don't want to spend too much time splitting hairs with frontend design, and I just want to focus on the backend stuff. (This was also my first time using AI for a project, so it was interesting)
